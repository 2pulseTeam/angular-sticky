Angular Sticky

Pure javascript AngularJS directive to make elements stick when scrolling

Code on Github Download

Getting started

This repository contains a set of native AngularJS directives and services. It can be used without any other dependency except of course Angular.

A few noteworthy things:

  • AngularJS (requires AngularJS 1.4.x or higher, tested with 1.4.9).
  • Small size: 16kB (~6kB with gzip compression!)

Installation

Bower

To install Angular Sticky with Bower use 

$ bower install angular-bootstrap

Manually

Click the download button to download the latest version of the plugin. In the folder dist you'll find the minified and normal JavaScript file. Either one is fine to use, it just depends on your needs.

Adding it to your project

As soon as you've got all the files downloaded and included in your page you just need to declare a dependency on the module: 

angular.module('myModule', ['hl.sticky']);

This will make an element sticky using the directive hl-sticky:

Features

Angular Sticky has been set up in such a way so it uses a lot of the handy features you're probably used to from Angular. A combination between between directives and services make the difference between all the other (native) sticky plugins. It allows you to have more control over the the look and feel of your sticky elements from both HTML and from code.

  • Directive: hlSticky

Directives

hlSticky

The magical sticky directive.

Attributes

  • hl-sticky/sticky-id (Default: none) - Determines the identifier of the sticky element.

    Most of the time you won't need this, but sometimes it's handy to set it. For example if you want to retrieve a sticky element from a stack. If this is not set, the identifier will default to the index it has in the stack.

  • media-query (Default: none) - Enables/disables the sticky element based on a media query.

    Check out the media query demo.
    Learn more about media queries.

  • sticky-class (Default: 'is-sticky') - Adds custom classes to the sticky element when it becomes sticky.
  • use-placeholder (Default: true) - Prevent the dom from collapsing on itself.

    When a sticky element becomes sticky its position becomes fixed. It is no longer part of the DOM in terms of other DOM element respecting it's dimensions and the elements underneath the sticky will suddenly jump a little. To compensate for this an invisible placeholder is created with the height of the sticky element.

  • anchor (Default: 'top') - Determines to what edge the sticky element should stick.
    • Note: This feature is under construction and only top can be used at this moment.
  • offset-top (Default: 0) - Adds offset to the top of the sticky element to act as margin.
    • Note: anchor must be set to top to make it work.
    • Note: This value can also be negative
  • offset-bottom (Default: 0) - Adds offset to the bottom of the sticky element to act as margin.
    • Note: This feature is under construction and cannot be used at this moment.
    • Note: anchor must be set to bottom to make it work.
    • Note: This value can also be negative
  • container (Default: None) - Id of a DOM element that you want the sticky element to be constrained to.

    Check out the container demo.
